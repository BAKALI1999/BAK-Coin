# BAK-Coin Telegram Bot 🪙

بوت تلجرم مجاني لتعدين وتوزيع عملة BAK-Coin

## الميزات

- ⛏️ تعدين مجاني للمستخدمين النشطين
- 💰 إرسال واستقبال عملات
- 📊 إحصائيات ومكافآت
- 🔗 نظام إحالات
- 🏆 لوحة صدارة

## الخطوات

### 1. إنشاء بوت تلجرم

```
1. افتح تلجرم
2. ابحث عن @BotFather
3. أرسل /newbot
4. اختر اسم: BAK-Coin Bot
5. اختر username: bakcoin_bot
6. انسخ Token
```

### 2. تثبيت المكتبات

```bash
pip install pyTelegramBotAPI
```

### 3. ضبط الإعدادات

افتح `bot_config.py` وضع Token:

```python
BOT_TOKEN = "YOUR_TOKEN_HERE"
```

### 4. تشغيل البوت

```bash
python run_bot.py
```

## أوامر البوت

| الأمر | الوصف |
|-------|-------|
| `/start` | بدء البوت |
| `/balance` | عرض الرصيد |
| `/mining` | بدء التعدين |
| `/daily` | مكافأة يومية |
| `/send @user amount` | إرسال عملات |
| `/refer` | رابط إحالة |
| `/leaderboard` | أقوى المعدّنين |
| `/stats` | إحصائيات |

## نظام التعدين

- كل مستخدم يبدأ التعدين بـ `/mining`
- يحصل على 10 BAK كل ساعة
- التعدين يعمل حتى لو أغلق البوت
- يمكن إيقاف التعدين بـ `/mining` مرة أخرى

## نظام الإحالات

- شارك رابط الإحالة
- كل صديق يسجل = 50 BAK مكافأة
- لا حد لعدد الإحالات

## المكافآت

- 🎁 مكافأة يومية: 5 BAK
- ⛏️ تعدين: 10 BAK/ساعة
- 🔗 إحالة: 50 BAK/صديق
- 💬 نشاط: 2 BAK/رسالة
