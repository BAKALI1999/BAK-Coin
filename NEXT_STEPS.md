# BAK-Coin — خطوات بعد النشر

## 1. إضافة سيولة على PancakeSwap (Testnet)

### ما تحتاجه:
```
- BAK tokens (عندك 21 مليون)
- TBNB للرسوم (من faucet)
```

### خطوات PancakeSwap:
```
1. افتح: https://testnet.pancakeswap.finance
2. اضغط "Connect Wallet" → MetaMask
3. اضغط "Trade" → "Liquidity"
4. اضغط "Add Liquidity"
5. اختر: BAK + TBNB
6. أضف السيولة
7. احصل على LP tokens
```

## 2. اختبار التحويل

### أرسل BAK لصديق:
```
1. افتح MetaMask
2. اضغط "Send"
3. أدخل عنوان صديقك
4. أدخل المبلغ
5. اضغط "Confirm"
6. انتظر التأكيد
```

## 3. النشر على Mainnet (قيمة حقيقية)

### ما تحتاجه:
```
- BNB (~$50) من Binance
- نفس الخطوات على Mainnet
```

### خطوات Mainnet:
```
1. أضف BSC Mainnet في MetaMask:
   - RPC: https://bsc-dataseed.binance.org/
   - Chain ID: 56
   - Symbol: BNB
   - Explorer: https://bscscan.com

2. نشر التوكن على Mainnet:
   - افتح Remix
   - غيّر الشبكة إلى BSC Mainnet
   - اضغط Deploy
   - وافق في MetaMask

3. أضف سيولة على PancakeSwap:
   - افتح: https://pancakeswap.finance
   - أضف BAK + BNB
   - حدد السعر
```
